# MyForcastApp
target_SDK: 11

## Architecture
MVVM-C

## Description
MyForcastApp permet de :
* Lister les détails des prévisions météo pour des zones/ pays introduits par l’utilisateur.
* Afficher des informations plus granulaires en sélectionnant un élément ajouté.
